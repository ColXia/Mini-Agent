## 4. 运行时路由流程图

```mermaid
flowchart TB
    Start["模型请求"]
    
    subgraph Resolution["路由解析"]
        ResolveCatalog["resolve_provider_catalog()"]
        RankRoutes["ProviderRouteSelector.rank()"]
        CheckBreaker["CircuitBreaker.should_allow()"]
    end
    
    subgraph Selection["候选选择"]
        AllowedCandidates["allowed_candidates"]
        BlockedCandidates["blocked_candidates"]
        SelectFirst["选择第一个允许的候选"]
    end
    
    subgraph Execution["执行"]
        CreateClient["创建 LLMClient"]
        CallAPI["调用 API"]
        RecordResult["记录结果"]
    end
    
    Start --> ResolveCatalog
    ResolveCatalog --> RankRoutes
    RankRoutes --> CheckBreaker
    CheckBreaker --> AllowedCandidates
    CheckBreaker --> BlockedCandidates
    AllowedCandidates --> SelectFirst
    SelectFirst --> CreateClient
    CreateClient --> CallAPI
    CallAPI --> RecordResult
    
    RecordResult --> |"成功"| Success["record_provider_success()"]
    RecordResult --> |"失败"| Failure["record_provider_failure()"]
    Failure --> |"允许故障转移"| NextCandidate["下一个候选"]
    Failure --> |"不允许故障转移"| Error["抛出异常"]
    NextCandidate --> CreateClient
```

    end
    
    Start --> GetBinding

    ResolveSkills --> ResolveMemory
    ResolveMemory --> PrepareTurn

    ParseResponse --> CheckToolCalls

    CheckToolCalls --> |"无工具调用"| CheckTransition

    ExecuteTool --> RecordMutation
    RecordMutation --> CollectResult
    CollectResult --> CheckTransition
    CheckTransition --> Continue
    CheckTransition --> Complete
    CheckTransition --> Failed
    CheckTransition --> Cancelled

## 8. 熔断器状态转换图

## 13. Agent 实体生命周期图

```mermaid
stateDiagram-v2
    [*] --> COLD: 创建 AgentInstance
    
    COLD --> READY: 初始化完成
    READY --> ATTACHED: 绑定 Workspace
    ATTACHED --> RUNNING: 启动 Run
    RUNNING --> WAITING: 等待用户输入/工具结果
    WAITING --> RUNNING: 收到响应
    RUNNING --> PAUSED: 用户暂停
    PAUSED --> RUNNING: 用户恢复
    RUNNING --> ATTACHED: Run 完成
    ATTACHED --> READY: 解绑 Workspace
    RUNNING --> ERRORED: 发生错误
    ERRORED --> READY: 错误恢复
    READY --> RETIRED: 实例退役
    ATTACHED --> RETIRED: 实例退役
    
    note right of COLD: 冷启动状态
    note right of READY: 就绪状态
    note right of ATTACHED: 已绑定工作空间
    note right of RUNNING: 执行中
    note right of WAITING: 等待中
    note right of PAUSED: 已暂停
    note right of ERRORED: 错误状态
    note right of RETIRED: 已退役
```
