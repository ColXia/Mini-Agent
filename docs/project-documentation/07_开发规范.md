## 2. 代码风格

### 2.1 Python 版本

- **最低版本**: Python 3.10+
- 使用 Python 3.10+ 新特性：
  - 类型联合语法 `str | None` 而非 `Optional[str]`
  - `match` 语句
  - `ParamSpec` 和 `TypeVarTuple`

### 2.2 格式化工具

- **格式化**: 使用 `ruff format`
- **Linting**: 使用 `ruff check`
- **类型检查**: 使用 `pyright` 或 `mypy --strict`

## 3. 命名约定

### 3.1 文件命名

- 模块文件：`snake_case.py`
- 测试文件：`test_{module_name}.py`
- 类型存根：`{module_name}.pyi`

### 3.2 类命名

- 类名：`PascalCase`
- 协议/接口：`{Name}Port` 或 `{Name}Protocol`
- 数据类：`{Name}` 或 `{Name}Entity`
- 枚举：`{Name}Enum` 或 `{Name}State`

```python
class AgentRuntimePort(Protocol): ...
class AgentInstance: ...
class RunStatus(str, Enum): ...
```

### 3.3 函数/方法命名

- 函数名：`snake_case`
- 异步函数：`async def`
- 私有方法：`_leading_underscore`

```python
async def list_agents(self) -> list[AgentSummary]: ...
def _internal_helper(self) -> None: ...
```

### 3.4 变量命名

- 变量名：`snake_case`
- 常量：`UPPER_SNAKE_CASE`
- 类属性：`snake_case`
- 实例属性：`snake_case`

```python
MAX_STEPS = 50
DEFAULT_TIMEOUT = 30.0

class Agent:
    max_steps: int = MAX_STEPS
    current_run_id: str | None = None
```

```python
async def managed_session(self, session_id: str):

    try:
        yield session
    finally:
        await self.cleanup_session(session_id)
```

## 7. 错误处理

### 7.1 自定义异常

```python
class MiniAgentError(Exception):
    """Base exception for Mini-Agent."""
    pass

class AgentNotFoundError(MiniAgentError):
    """Agent not found."""
    def __init__(self, agent_id: str):
        self.agent_id = agent_id
        super().__init__(f"Agent not found: {agent_id}")

class RunExecutionError(MiniAgentError):
    """Run execution failed."""
    def __init__(self, run_id: str, reason: str):
        self.run_id = run_id
        self.reason = reason
        super().__init__(f"Run {run_id} failed: {reason}")
```

### 7.2 错误处理模式

```python
async def execute_run(self, run_id: str) -> RunResult:
    try:
        run = await self._get_run(run_id)
        result = await self._execute(run)
        return result
    except AgentNotFoundError as e:
        logger.error(f"Agent not found: {e.agent_id}")
        raise
    except RunExecutionError as e:
        logger.error(f"Run execution failed: {e.run_id}")
        await self._handle_run_failure(run_id, e.reason)
        raise
    except Exception as e:
        logger.exception(f"Unexpected error in run {run_id}")
        raise RunExecutionError(run_id, str(e)) from e
```

### 8.2 日志级别

| 级别 | 使用场景 |
|------|----------|
| `DEBUG` | 详细调试信息 |
| `INFO` | 正常操作信息 |
| `WARNING` | 警告信息，不影响运行 |
| `ERROR` | 错误信息，需要关注 |
| `CRITICAL` | 严重错误，影响运行 |

### 8.3 日志格式

```python
# 正确：结构化日志
logger.info(
    "Run started",
    extra={
        "run_id": run_id,
        "agent_id": agent_id,
        "session_id": session_id,
    }
)

# 正确：简单日志
logger.info(f"Run {run_id} started")

# 错误：敏感信息
logger.info(f"API key: {api_key}")  # 避免记录敏感信息
```

## 12. 安全规范

### 12.1 敏感信息处理

- 不在日志中记录 API 密钥、密码等敏感信息
- 使用环境变量存储敏感配置
- 敏感数据传输使用加密

### 12.2 输入验证

- 所有外部输入进行验证
- 使用 Pydantic 进行 API 边界验证
- 文件路径检查边界

### 12.3 权限控制

- 遵循最小权限原则
- 危险操作需要审批
- 记录所有变更操作
