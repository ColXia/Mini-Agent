## 2. 版本历史

### v11.1 (2026-05-10)

**架构变更**

- 硬切分 Runtime 模块，明确职责边界
- 硬切分 Package Facades，简化服务门面
- 硬切分 Commands Router，统一命令路由

**新增功能**

- 新增 `SessionTaskService` 管理 Session-Run 映射
- 新增 `UserServiceAssembly` 显式组装用户服务
- 新增 `ManagedSessionTurn` 封装会话回合执行

**改进**

- 重构 Application Layer 依赖注入
- 优化 Port 定义，遵循接口隔离原则
- 改进事件系统，支持异步事件处理

**移除**

- 移除 `interaction` 模块（已整合到其他模块）
- 移除 `root sdk` shims（不再需要兼容层）

## 4. 版本号规则

遵循语义化版本规范 (SemVer):

- **主版本号 (MAJOR)**: 不兼容的 API 变更
- **次版本号 (MINOR)**: 向后兼容的功能新增
- **修订号 (PATCH)**: 向后兼容的问题修复

示例: `v11.1.0`
- `11`: 主版本号
- `1`: 次版本号
- `0`: 修订号

## 7. 迁移指南

### 7.1 v10.x → v11.x

**破坏性变更**

- `AgentService` 重构为 `AgentUserService`
- `WorkspaceService` 重构为 `WorkspaceUserService`
- Port 定义位置变更

**迁移步骤**

1. 更新服务导入路径
2. 使用 `UserServiceAssembly` 组装服务
3. 更新依赖注入配置

### 7.2 v9.x → v10.x

**破坏性变更**

- Context 组装方式变更
- Skill 解析接口变更

**迁移步骤**

1. 使用 `ContextAssembler` 替代直接组装
2. 更新技能名称前缀
3. 使用 `SkillResolver` 解析技能
