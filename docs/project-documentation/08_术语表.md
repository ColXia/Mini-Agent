# Mini-Agent 术语表

## 1. 概述

本文档定义 Mini-Agent 项目中使用的术语和缩写，确保团队成员对概念有统一理解。

---

## 2. 核心概念

### 2.1 Agent 相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Agent | Agent | AI 代理，能够理解指令、调用工具、执行任务的智能实体 |
| Agent Profile | Agent Profile | Agent 的静态配置，包括系统提示、工具策略、技能偏好等 |
| Agent Instance | Agent Instance | Agent 的运行时实例，有生命周期状态 |
| Agent Kernel | Agent Kernel | Agent 的核心构建器，负责组装 Agent 的各个组件 |
| Agent Engine | Agent Engine | Agent 的执行引擎，负责执行循环和工具调用 |

### 2.2 Run 相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Run | Run | 一次正式的执行单元，从用户输入到最终响应的完整过程 |
| Turn | Turn | 单轮对话，一个 Run 可能包含多个 Turn |
| Step | Step | 执行步骤，一次 LLM 调用或工具调用 |
| Checkpoint | Checkpoint | 执行状态快照，用于恢复和回滚 |
| Phase | Phase | Run 的执行阶段，如 CREATED、BINDING、EXECUTING_TOOLS 等 |

### 2.3 Session 相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Session | Session | 会话，管理对话历史和状态 |
| Message | Message | 消息，会话中的单条消息 |
| Content Block | Content Block | 内容块，消息中的结构化内容单元 |
| Attachment | Attachment | 附件，消息携带的文件 |

### 2.4 Workspace 相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Workspace | Workspace | 工作空间，定义 Agent 的操作边界和权限 |
| Boundary | Boundary | 边界，工作空间的访问限制 |
| Permission Table | Permission Table | 权限表，定义工作空间的权限规则 |
| Mutation | Mutation | 变更，工作空间内的修改操作 |
| Mutation Ledger | Mutation Ledger | 变更账本，记录所有变更操作 |
| Snapshot | Snapshot | 快照，工作空间状态的存档 |

---

## 3. 架构术语

### 3.1 分层架构

| 术语 | 英文 | 定义 |
|------|------|------|
| Surface Layer | Surface Layer | 表面层，用户界面层（CLI、Desktop、Gateway） |
| Application Layer | Application Layer | 应用层，用例编排和端口定义 |
| Agent Core Layer | Agent Core Layer | 核心层，Agent 执行引擎 |
| Service Layer | Service Layer | 服务层，领域服务 |
| Infrastructure Layer | Infrastructure Layer | 基础设施层，外部依赖实现 |

### 3.2 六边形架构

| 术语 | 英文 | 定义 |
|------|------|------|
| Port | Port | 端口，定义接口契约的 Protocol |
| Adapter | Adapter | 适配器，实现 Port 的具体类 |
| Inbound Adapter | Inbound Adapter | 入站适配器，接收外部请求（CLI、Gateway） |
| Outbound Adapter | Outbound Adapter | 出站适配器，调用外部服务（LLM Client） |

### 3.3 服务类型

| 术语 | 英文 | 定义 |
|------|------|------|
| Use Case Service | Use Case Service | 用例服务，编排业务逻辑 |
| Facade Service | Facade Service | 门面服务，简化复杂流程 |
| User Service | User Service | 用户服务，对外暴露的服务门面 |
| Application Service | Application Service | 应用服务，应用层的业务逻辑 |
| Domain Service | Domain Service | 领域服务，核心业务逻辑 |

## 4. 模型管理术语

### 4.1 提供商相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Provider | Provider | 模型提供商，如 Anthropic、OpenAI |
| Provider Catalog | Provider Catalog | 提供商目录，所有提供商的注册表 |
| Provider Route | Provider Route | 提供商路由，模型到提供商的映射 |
| Provider Source | Provider Source | 提供商来源类型，如 anthropic、openai、openrouter |

### 4.2 路由相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Route Selector | Route Selector | 路由选择器，选择最佳提供商路由 |
| Model Pool | Model Pool | 模型池，管理所有可用模型 |
| Model Binding | Model Binding | 模型绑定，Agent 与模型的关联 |
| Model Alias | Model Alias | 模型别名，模型的友好名称 |

### 4.3 熔断相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Circuit Breaker | Circuit Breaker | 熔断器，防止级联故障的保护机制 |
| Closed State | Closed State | 正常状态，允许所有请求 |
| Open State | Open State | 熔断状态，拒绝所有请求 |
| Half-Open State | Half-Open State | 半开状态，允许探测请求 |
| Failover | Failover | 故障转移，自动切换到备用提供商 |

### 4.4 能力相关

| 术语 | 英文 | 定义 |
|------|------|------|
| Model Capabilities | Model Capabilities | 模型能力，模型支持的功能特性 |
| Capability Detection | Capability Detection | 能力探测，自动检测模型能力 |
| Vision Support | Vision Support | 视觉支持，模型是否支持图像输入 |
| Tool Support | Tool Support | 工具支持，模型是否支持函数调用 |
