# Mini-Agent 数据合约

## 1. 概述

本文档定义 Mini-Agent 的核心数据合约（Contracts）。所有合约使用 `@dataclass(frozen=True, slots=True)` 定义，确保不可变性和内存效率。

---

## 2. Agent 合约

### 2.1 AgentProfile

```python
@dataclass(frozen=True, slots=True)
class AgentProfile:
    """Agent 静态配置 - 持久化"""
    agent_profile_id: str
    name: str
    description: str = ""
    system_prompt: str = ""
    default_model_preference: str | None = None
    tool_policy: dict[str, Any] = field(default_factory=dict)
    skill_preferences: dict[str, Any] = field(default_factory=dict)
    max_steps: int = 50
    max_tool_calls_per_step: int | None = None
    token_limit: int = 80000
    metadata: dict[str, Any] = field(default_factory=dict)
```

## 12. 设计原则

### 12.1 不可变性

- 所有合约使用 `frozen=True`
- 状态变更通过创建新实例
- 避免副作用和意外修改

### 12.2 内存效率

- 使用 `slots=True` 减少内存占用
- 使用 `frozenset` 替代 `set`
- 大型集合使用 `tuple` 替代 `list`

### 12.3 序列化友好

- 所有字段支持 JSON 序列化
- 使用基本类型和容器
- 避免复杂的嵌套结构

### 12.4 版本兼容

- 新字段提供默认值
- 使用 `field(default_factory=...)` 处理可变默认值
- 保持向后兼容性
