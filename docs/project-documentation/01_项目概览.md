# Mini-Agent 项目概览

## 1. 项目简介

**Mini-Agent** 是一个智能 Agent 平台，提供 CLI、Gateway 和可扩展子程序支持。

### 1.1 项目定位

Mini-Agent 是一个 **AI Agent 开发框架**，核心特性：

- **多模型支持**: 支持 Anthropic、OpenAI、OpenRouter、Ollama 等多种 LLM 提供商
- **技能系统**: 可扩展的技能 (Skills) 机制，支持自定义能力扩展
- **工具调用**: 内置工具系统，支持 MCP (Model Context Protocol) 协议
- **工作空间**: 支持多工作空间管理和会话持久化
- **Gateway API**: 提供 HTTP API 供外部系统集成
- **桌面应用**: 支持 PySide6 桌面客户端

### 1.2 技术栈

| 类别 | 技术 |
|------|------|
| 语言 | Python 3.10+ |
| 异步框架 | asyncio |
| HTTP 框架 | FastAPI + Uvicorn |
| LLM SDK | anthropic, openai |
| 数据验证 | Pydantic 2.0 |
| 终端 UI | prompt-toolkit |
| 桌面 UI | PySide6 (可选) |
| MCP 协议 | mcp >= 1.0.0 |
| Token 计算 | tiktoken |

### 1.3 版本信息

- **当前版本**: v0.2.0
- **架构版本**: v11.1
- **许可证**: MIT

## 3. 核心模块概述

### 3.1 Agent Core (agent_core/)

Agent 核心引擎，负责：
- Agent 生命周期管理
- 执行循环和工具调用
- 上下文管理和压缩
- 会话和检查点
- 技能加载和执行

**关键文件**:
- `engine.py` - Agent 引擎核心
- `kernel.py` - Agent 内核
- `contracts/` - 核心数据合约

### 3.2 Application Layer (application/)

应用服务层，负责：
- 用例编排
- 端口定义 (Ports)
- 门面服务 (Facades)

**关键文件**:
- `ports/` - 运行时端口定义
- `use_cases/` - 应用服务
- `facades/` - 门面服务

### 3.3 Model Manager (model_manager/)

模型管理器，负责：
- Provider 注册和配置
- 模型选择和路由
- 熔断器和健康监控
- 能力探测

**关键文件**:
- `model_pool.py` - 模型池
- `runtime.py` - 运行时路由
- `agent_model_service.py` - Agent 模型服务

### 3.4 LLM Clients (llm/)

LLM 客户端实现，负责：
- 协议适配 (Anthropic/OpenAI)
- 请求构建和响应解析
- 流式输出处理

**关键文件**:
- `base.py` - 客户端基类
- `anthropic_client.py` - Anthropic 实现
- `openai_client.py` - OpenAI 实现

### 3.5 Runtime (runtime/)

运行时服务，负责：
- 请求处理
- 会话编排
- 实时控制

**关键文件**:
- `handlers/` - 请求处理器
- `orchestration/` - 编排服务
- `live_control/` - 实时控制

### 3.6 Workspace & Session

工作空间和会话管理，负责：
- 工作空间边界
- 会话持久化
- 附件管理

**关键文件**:
- `workspace_runtime/` - 工作空间运行时
- `session/` - 会话存储

### 3.7 Tools & Skills

工具和技能系统，负责：
- 内置工具注册
- 技能加载和执行
- MCP 协议支持

**关键文件**:
- `tools/` - 工具定义
- `skills/` - 技能目录

## 4. 入口点

### 4.1 CLI 入口

```bash
# 主命令
mini-agent

# 简短别名
mini

# 交互式 CLI
mini-agent-cli
```

### 4.2 Gateway 入口

```bash
# 启动 Gateway 服务
uvicorn mini_agent.gateway:app --host 0.0.0.0 --port 8000
```

### 4.3 桌面应用入口

```bash
# 启动桌面应用
python -m mini_agent.desktop
```

## 5. 配置

### 5.1 环境变量

| 变量 | 说明 |
|------|------|
| `ANTHROPIC_API_KEY` | Anthropic API 密钥 |
| `OPENAI_API_KEY` | OpenAI API 密钥 |
| `MINI_AGENT_PROVIDER_CATALOG_PATH` | Provider 配置文件路径 |

### 5.2 配置文件

| 文件 | 位置 | 说明 |
|------|------|------|
| `providers.json` | `~/.mini-agent/` | Provider 配置 |
| `agent_model_binding.json` | `~/.mini-agent/` | Agent 模型绑定 |
| `.env` | 项目根目录 | 环境变量 |
