# Archived Builtin Skills

These skills were previously present under `src/mini_agent/skills/` as example or legacy bundled skills.

They have been moved here so that:

- Mini-Agent's default builtin bundle stays aligned with the active product direction
- the old examples remain available for reference
- the runtime skill loader no longer exposes them by default

Archived entries in this directory are reference material only unless they are manually copied into a workspace skill directory.
